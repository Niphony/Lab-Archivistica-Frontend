## Inicio rapido

### Requisitos

- Node.js 20 o superior (recomendado 22 LTS) - versión usada 26.1.0
- npm 10 o superior 

### 1) Instalar dependencias

```bash
npm ci
```

### 2) Ejecutar en desarrollo - Para tener preview en localhost

```bash
npm run dev
```

### 3) Build de produccion

```bash
npm run build 
```

### 4) Preview local

```bash
npm run preview
```
